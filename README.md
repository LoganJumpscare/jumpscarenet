# LoganJumpscare

A collection of spooky animations and mini-games built entirely with HTML and JavaScript for entertainment purposes.

## About

JUMPSCARE.NET is a fun, retro-themed terminal experience featuring jump scares and interactive games. All content runs locally in your browser with no data collection or tracking.

## Current Games

- **Temu Granny** - A mysterious interactive experience

## Future Games

More animations and mini-games coming soon! This project will continue to expand with new spooky entertainment.

## Features

- Pure HTML & JavaScript (no frameworks)
- Retro terminal aesthetic with CRT effects
- Privacy-focused (no data collection)
- Responsive design
